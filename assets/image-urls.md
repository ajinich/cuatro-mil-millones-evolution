## How to Download Images

### Option 1: Manual Download
1. Right-click each URL above
2. Save to the `assets/images/` folder
3. Keep original filenames for consistency

### Option 2: Script Download
Use a script like `wget` or `curl` to download all images:

```bash
# Create images directory
mkdir -p assets/images

# Download key images (examples)
curl -o assets/images/P_014_015_arbol_evol.jpg "https://images.squarespace-cdn.com/content/v1/646296151dd4540600b395ab/92419bd5-1576-4e50-8175-3a8f37cf431e/P_014_015_arbol_evol.jpg"
curl -o assets/images/darwincito.jpg "https://images.squarespace-cdn.com/content/v1/646296151dd4540600b395ab/508b53f4-b4f3-4f28-a89e-4af3c020166b/darwincito.jpg"
```

### Option 3: Browser Extension
Use a browser extension that can bulk download images from a webpage.
